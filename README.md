# Sports-Event-Management-MySQL

# Sports Event Management System (MySQL)

## Description
A pure MySQL-based DBMS project to manage sports events, teams, players,
matches, sponsors, and results.

## Technologies Used
- MySQL 8.0
- SQL

## File Structure
- 01_schema.sql – Database & table creation
- 02_procedures_functions.sql – Procedures & functions
- 03_triggers.sql – Triggers
- 04_sample_data.sql – Sample data
- 05_queries.sql – Queries

## How to Run
1. Open MySQL Workbench
2. Execute files in order from 01 → 05

## Concepts Used
- Normalization
- Constraints
- Foreign Keys
- Triggers
- Stored Procedures
- Functions
- Joins
